Weather app

For the following assignment, I have created a simple Weather app in which you can search the name of a city and check the temperature. For this assignment, I have used the OpenWeatherMap API to retrieve the current temperature information. The JavaScript code begins with the declaration of two variables: apiKey and apiURL. These variables store the API key and the URL of the OpenWeatherMap API. The checkWeather(city) function is defined to fetch weather data for a given city from the OpenWeatherMap API using fetch(). It uses the await keyword to wait for the response and then processes the returned data. If the API responds with a status of 404 (city not found), the error message is displayed, and the weather information is hidden. Otherwise, the weather data is extracted from the response JSON and displayed in the corresponding HTML elements. The weather image is updated based on the weather condition. The searchBtn element listens for a click event, and when clicked, it calls the checkWeather() function with the value of the search input box.

USD to BGN Converter

For this assignment, I have created a function that allows me to convert an amount in USD to BGN. This assignment doesn't consist of any additional webpage. Inside the function, the variable usd is assigned the value of the first element in the amount array, which is accessed using amount[0]. The variable bgn is then assigned the result of multiplying the usd value (converted to a number using Number()) by the exchange rate of 1.79549 (presumably from USD to BGN, Bulgarian Lev). Finally, the bgn value is logged to the console.
